3D Kane-Mele Model with Anderson Disorder

Repositori ini berisi kumpulan skrip simulasi numerik menggunakan PythTB 2.0 untuk mempelajari fase topologis pada model Kane-Mele 3D (Stacked Graphene). Fokus utama riset ini adalah menyelidiki ketangguhan edge states terhadap pengaruh Rashba Coupling, Staggered Potential, dan Anderson Disorder.
